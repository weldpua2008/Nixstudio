Что такое KNOPPIX®?

KNOPPIX - это собрание готовых программ GNU/Linux, работающих прямо с CD-ROM. Компоненты вашего компютера распознаются автоматически: многие видео- и звуковые карты, SCSI контроллеры и другое оборудование. KNOPPIX можно использовать как демонстрационный CD, для обучения или для восстановления системы, или же как основу для разработки новых коммерческих продуктов. Установка на жесткий диск не требуется. С использованием прозрачной декомпрессии, на однин компкакт-диск входит до двух гигабайт программного обеспечения (на этом диске около 1900Мб).

Так как это настоящий open source проект, он расространяется на условиях Стандартной Общественной Лицензии GNU (GNU General Public License), исходный код для KNOPPIX-специфичных пакетов доступен в Internet на http://www.knopper.net/knoppix/sources/. Исходники для установленых Debian пакетов вы можете найти на различных зеркалах Debian. Также, вы можете заказать иходный код прямо с Knopper.Net по цене материала, копирования, упаковки и отправки почтой. Это предложение действительно 3 года, начаная с даты выпуска этого CD-Rom'а.

Системные требования

  • Intel-совместимый CPU (i486 или новее),
  • 32 MB RAM для текстового режима, и хотя бы 82 MB(swap-раздел+ОЗУ) для графического режима в KDE. (Для запуска различных оффисных програм, рекомендовано хотя бы 128 MB),
  • загрузочный CD-ROM/DVD (IDE/ATAPI, Firewire, USB or SCSI),
  • стандартная SVGA-совместимая видео карта,
  • стандартная serial или PS/2 мышь, или IMPS/2-совместимая USB мышь.

Вопрос: "На моём компьютере нет достаточного количества оперативной памяти (RAM), поэтому я не могу запустить некоторые из программ на CD (или они запускаются очень медленно). есть ли способ запустить KDE и офисные приложения, видео, игры?"

ДА. Использованием раздела подкачки (swap) недостающая RAM под Linux может быть заменена местом на жестком диске, зарезервированым для этих целей. KNOPPIX найдет и автоматически начнет использовать доступные разделы swap. Однако только опытному прользователю стоит заниматься созданием swap раздела, потому что для его создания нужно переразметить жесткий диск. Начиная с версии 1.5 KNOPPIX'a "swapfile" для Linux также может быть создан на существующем разделе DOS (команда mkdosswapfile). Использование swap позволяет KNOPPIX запускаться с меньшим количеством доступной оперативной памяти. Файл созданый в DOS разделе (knoppix.swp) может быть позже удален, если место понадобится для каких-то других целей.

На http://www.knopper.net/knoppix/ вы найдете несколько зеркал в Internet, которые, в определенной мере, предоставляют к загрузке текущий KNOPPIX ISO образ.
Вы также можете заказать версию адаптированую к вашим требованиям с помощью Контактной формы на Knopper.Net.

Как запустить KNOPPIX?

Для того чтобы загрузится с CD, в первую очередь настройте boot list в настройках BIOS вашего компьтера на загрузку с CD (старт перед жестким диском). Вставьте CD и перезагрузите компьютер. Если ваш компьютер не поддерживает данный режим, вам нужно использовать загрузочную дискету, которую вы можете создать с помощью загрузочного образа /KNOPPIX/boot.img доступного на CD. Используйте скрипт mkfloppy.bat Для стандартного загрузочного образа (Немецкая версия). Используйте команду rawrite2 -f boot-en.img -d A (в директории KNOPPIX) Для создания загрузочного диска на Английском языке.

Какой софт присутствует на KNOPPIX-CD?

В добавок к стандартному Debian-based GNU/Linux дистрибутиву (www.debian.org) на этом CD присутствуют:

  • KDE V3.x как стандартный desktop, с KOffice и www-броузером Konqueror.
  • X Multimedia System (xmms) с MPEG видео и MP3 поддержкой также как и с Ogg Vorbis Audio Player'ом
  • Программы для доступа в Internet kppp и isdn-utilities
  • Gnu Image Manipulation Program (GIMP) Version 2.0
  • Инструменты для восстановления информации и системы
  • Инструменты для анализа сети и администрирования
  • OpenOffice(TM) 1.1.x, GPL версия известного StarOffice(TM) оффисного пакета
  • Множество языков программирования, средств разработки,, так же как и библиотеки для разработчиков.
  • В общей сложности, более 900 установленных пакетов, охватывают более 2000 программ, утилит, и игр.
Для более детальной информации о текущих изменениях смотрите обзор установленных пакетов и CHANGELOG. Номер версии (=дата build'a) вашего CD-Rom показана на загрузочном экране в правом верхнем углу. Пожалуйста всегда указывайте этот номер, когда у вас есть вопрос или комментарий.

Tips & Tricks

Здесь собраны некоторые советы (объединенные в один файл Knoppix Cheatcodes) Описывающие опции, которые могут быть переданы Knoppix, чтобы разрешать специфические ситуации и/или случаи с проблемным "железом". Некоторые из них требуют более детального знания ОС Linux, другие же просто полезные дополнения автоконфигуратора KNOPPIX.
